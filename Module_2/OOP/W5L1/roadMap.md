# Object-Oriented Programming - Module 2, Week 5, Lesson 1
## Learning Roadmap

### Course Outline

1. **Introduction to Inheritance**
   - Definition and purpose of inheritance
   - Base classes and derived classes
   - Access specifiers (public, private, protected)
   - Single inheritance concepts

2. **Polymorphism**
   - Run-time Polymorphism (function overriding)
   - Compile-time Polymorphism (function overloading)
   - Benefits and use cases
   - Implementation strategies

3. **Virtual Functions**
   - Virtual function declaration and usage
   - Dynamic binding mechanism
   - Pure virtual functions and abstract classes
   - Virtual function tables (VTables)

4. **Advanced Inheritance Concepts**
   - Multiple Inheritance (inheriting from multiple base classes)
   - Multilevel Inheritance (chain of inheritance)
   - Practical code examples and demonstrations
   - Common pitfalls and solutions

5. **Friend Functions**
   - Friend function declaration and implementation
   - Breaking encapsulation when necessary
   - Use cases and best practices

### Learning Objectives

By the end of this lesson, students will understand:
- How inheritance enables code reusability
- The difference between polymorphism types
- When and how to use virtual functions
- How multiple and multilevel inheritance work
- How to apply these concepts in real-world scenarios
